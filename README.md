# rmpv

A minimal MPD + MPV + YouTube bridge for Linux.

## Features
- Play local MPD songs via mpv
- Stream YouTube instantly
- Fast yt-dlp search
- CLI-based workflow
- Works with rmpc

## Install
```bash
git clone https://github.com/YOURNAME/rmpv
cd rmpv
bash scripts/install.sh
